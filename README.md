## Project Description

Campus Life App is a web application designed for students of **Blue Mountain Christian University**.  
It provides quick access to campus resources, including events, dining options, campus maps, and live weather information. The app emphasizes accessibility, responsiveness, and user-friendly navigation.

---

## Features

- **Home Page**
  - Campus hero image
  - Quick navigation to Events and Dining pages
  - Live weather display for Blue Mountain, MS

- **Events Page**
  - Displays the BMC Sports Calendar
  - Reminds students to check their BMC email for additional events

- **Dining Page**
  - Ray Dining Hall and Topper Market (weekdays only)
  - Detailed hours for each location
  - Images for each dining location
  - Direct link to [BMC Dining](https://www.bmc.edu/dining)

- **Map Page**
  - Displays a campus map image (`map.jpg`)
  - Full-page responsive display for easy viewing

- **Navigation**
  - Consistent navbar across all pages: Home, Events, Dining, Map
  - Responsive design using Bootstrap

- **Accessibility**
  - Semantic HTML headings
  - Alt text for all images
  - Color contrast suitable for readability

## Technologies Used

- HTML5 & Semantic HTML
- CSS3, including custom styles and media queries
- JavaScript (vanilla) for dynamic interactivity
- Bootstrap 5.3 for responsive layout and components
- OpenWeatherMap API for live weather
